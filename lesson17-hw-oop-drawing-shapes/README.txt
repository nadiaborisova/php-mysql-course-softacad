Домашна работа към лекция 17 - част 2:

Class Point:
Да се създаде клас Point, който да представлява точка с x,y координати. Да има всички нужни setter-и и getter-и, да има конструктор и да има функция, която позволява преместването на точката.
Пример за преместване: Ако точката ни е с координати 4,5 и я преместим с 1,1 то тя трябва да се намира на 5,6 след това.

Class Line:
Да се създаде клас Line, който да представлява отсечка. Отсечката се характеризира с точка за начало и точка за край. За тези две точки да се използва тип Point (от предишната задача).
Отсечката да има конструктор, приемащ 2 x Point, както и всички getter-и и setter-и. Да може да се мести по същия начин, както се местеха точките. При преместване на отсечката 2-те точки се местят с еднакви стойности.
Да се създаде метод, който да смята дължината на отсечката. За помощ: http://www.regentsprep.org/regents/math/geometry/gcg3/ldistance.htm

Class Triangle:
Да се създаде клас Triangle който да представлява триъгълник. Той се характеризира с три точки. За тези три точки да се използва тип Point (от предишната задача).
Триъгълникът да има конструктор, приемащ 3 x Point, както и всички getter-и и setter-и. Да може да се мести по същия начин, както се местеха точките. При преместване на отсечката 3-те точки се местят с еднакви стойности.
Да има метод, който да смята лицето на триъгълника. За помощ: http://www.mathopenref.com/heronsformula.html

Class Shape:
Да се създаде клас Shape който да представлява произволен многоъгълник. Той се характеризира с N точки. За тези N точки да се използва тип Point (от предишната задача).
Класът Shape да има конструктор, приемащ произволен брой параметри.
Класът Shape да има getter, връщащ избрана точка (избираме я по index / id).
Класът Shape да има setter, модифициращ избрана точка (избираме я по index / id).
Класът Shape да има метод remove, премахващ избрана точка (избираме я по index / id).
Класът Shape да има метод add, добавящ нова точка (добавяме я като последна).
Да има метод, който връща обиколката на многоъгълника, следвайки реда на точките в него.
